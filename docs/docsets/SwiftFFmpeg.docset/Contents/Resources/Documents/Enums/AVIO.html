  • realloc(_:size:)

    Allocate, reallocate, or free a block of memory.

    If ptr is nil and size > 0, allocate a new block. If size is zero, free the memory block pointed to by ptr. Otherwise, expand or shrink that block of memory according to size.

    Warning

    Unlike malloc, the returned pointer is not guaranteed to be correctly aligned.

    Declaration

    Swift

    public static func realloc(_ ptr: UnsafeMutableRawPointer?, size: Int) -> UnsafeMutableRawPointer?

    Parameters

    ptr

    Pointer to a memory block already allocated with realloc or nil
    size

    Size in bytes of the memory block to be allocated or reallocated

    Return Value

    Pointer to a newly-reallocated block or nil if the block cannot be reallocated or the function is used to free the memory block

  • fileMap(filename:buffer:size:)

    Read the file with name, and put its content in a newly allocated buffer or map it with mmap() when available. In case of success set buffer to the read or mmapped buffer, and size to the size in bytes of the buffer. Unlike mmap this function succeeds with zero sized files, in this case *bufptr will be set to nil and *size will be set to 0. The returned buffer must be released with fileUnmap.
